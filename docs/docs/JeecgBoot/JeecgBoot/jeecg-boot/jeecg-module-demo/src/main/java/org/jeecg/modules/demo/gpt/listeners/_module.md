# 说明

OpenAISSEEventSourceListener类负责处理SSE（Server-Sent Events）事件，主要功能包括管理消息内容的接收与解析、执行回调函数以响应事件，以及处理可能发生的错误。该类确保事件流的稳定性和可靠性，通过有效的错误处理机制保障系统的正常运行。
