# 说明

JeecgEntity类是一个基础实体类，包含五个核心字段：ID用于唯一标识实体，创建人记录实体的创建者，创建时间记录实体的创建时间，更新人记录实体的最后修改者，更新时间记录实体的最后修改时间。这些字段有助于跟踪实体的生命周期和变更历史。
