# 说明

OssBootUtil类是一个用于管理阿里云OSS文件操作的工具类，主要功能包括文件上传、删除以及获取文件流等。该类支持用户自定义存储桶名称，并允许通过文件路径进行过滤，以便更灵活地处理OSS中的文件。通过OssBootUtil，用户可以方便地执行与OSS相关的各种操作，提升文件管理的效率和便捷性。
